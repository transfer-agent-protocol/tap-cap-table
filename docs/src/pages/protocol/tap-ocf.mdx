# Open Cap Table Format

The Open Cap Table Format ([OCF](https://github.com/transfer-agent-protocol/tap-ocf)) is a JSON-based open-source standard for storing and sharing cap table data. We use OCF to store data off chain. The format is maintained by the [Open Cap Table Coalition](https://www.opencaptablecoalition.com/). You'll find the OCF modules in the root folder:

## OCF Terminology

The Open Cap Format uses specific terminology:

- **Stakeholder**: An equity holder on the cap table (shareholder). Can be an individual (person) or institution (company/fund).
- **Issuer**: The company whose cap table is being tracked.
- **Security**: A specific equity position (shares, options, warrants, convertibles).
- **Transaction**: Events that modify the cap table (issuances, transfers, cancellations, etc.).

Note that "stakeholder" in OCF context is more specific than the general business termâ€”it refers only to equity holders, not all parties with interest in the company.
