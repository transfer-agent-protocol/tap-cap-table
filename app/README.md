# TAP Frontend

Next.js frontend for the [Transfer Agent Protocol](https://transferagentprotocol.xyz).

## Development

```bash
# From monorepo root
pnpm app:dev

# Or from this directory
pnpm dev
```

## Tech Stack

- Next.js (Pages Router)
- styled-components
- IBM Plex Mono font

See the root [CONTRIBUTING.md](../CONTRIBUTING.md) for contribution guidelines.
